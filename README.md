# Python Fundamentals Labs

This repository contains a collection of hands-on Python labs designed for beginners to intermediate learners. The labs focus on practical programming skills commonly used in Data Analytics, Data Engineering, AI, ETL/ELT workflows, automation, and general software development.

Each lab includes:

* Learning objectives
* Step-by-step instructions
* Sample code
* Expected outputs
* Challenge exercises
* Knowledge checks

---

## Prerequisites

Before starting these labs, you should have:

* Python 3.x installed
* Visual Studio Code (or another Python IDE)
* Basic familiarity with using the command line

---

## Lab Overview

| Lab    | Topic                                    | Key Concepts Covered                                                        |
| ------ | ---------------------------------------- | --------------------------------------------------------------------------- |
| Lab-1  | Variables and Literal Types              | Variables, integers, floats, booleans, `print()`, `type()`                  |
| Lab-2  | Numeral System Conversions               | Binary, hexadecimal, decimal, `bin()`, `hex()`, `int()`                     |
| Lab-3  | Encoding and Decoding Data               | ASCII values, `ord()`, `chr()`, hexadecimal encoding, decoding              |
| Lab-4  | Refactoring Variables Using PEP 8        | Naming conventions, code readability, variable refactoring                  |
| Lab-5  | Clear Naming in Data Pipelines           | Descriptive naming, transformation scripts, ETL-style coding practices      |
| Lab-6  | Structured User Input                    | `input()`, `float()`, collecting temperature, distance, and currency values |
| Lab-7  | Output Formatting and Data Logging       | `print()`, `sep=`, `end=`, log formatting, reporting                        |
| Lab-8  | Type Casting for Calculations            | `int()`, `float()`, `bool()`, `complex()`, user input conversion            |
| Lab-9  | Input Validation and Error Handling      | `try`, `except`, validation checks, retry loops, defensive programming      |
| Lab-10 | Unit Converter with CSV and JSON Logging | Unit conversion, structured output, CSV logging, JSON logging               |
